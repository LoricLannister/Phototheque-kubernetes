## Photothèque : Kubernetes Deployment Pipeline

Le projet Photothèque est une application web développée en Node.js, déployée sur un cluster Kubernetes local.

L’application permet de gérer une photothèque en ligne : elle stocke, sert et met en cache des images à travers un ensemble de services interconnectés.
Derrière cette logique applicative, tout le travail du projet repose sur la mise en place d’une infrastructure distribuée et automatisée.

## Objectifs :

Ce projet a pour vocation de :

maîtriser les concepts de conteneurisation et d’orchestration avec Kubernetes,

mettre en œuvre une infrastructure automatisée et scalable,

apprendre à documenter une architecture technique de manière claire et professionnelle,

et simuler une pipeline de déploiement continue (CI/CD) complète en environnement de développement.
